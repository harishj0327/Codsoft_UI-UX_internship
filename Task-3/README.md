# 🍝 Task 3 – Restaurant Mobile App UI

## 📖 Project Overview

This project is a modern restaurant mobile application designed using **Figma** as part of the **CodSoft UI/UX Design Internship**.

The application provides a smooth and user-friendly food ordering experience with attractive visuals, intuitive navigation, and interactive prototypes.

## 🍝 Project Name

**Pasta Palace – Italian Restaurant App**

## 📱 Screens Included

- Splash Screen
- Home Screen
- Food Details Screen
- Add to Cart Success Popup

## ✨ Features

- Modern & Clean UI Design
- Mobile-First Approach
- Interactive Prototype
- Food Search Functionality
- Category-Based Navigation
- Add to Cart Experience
- User-Friendly Layout

## 🛠️ Tools Used

- Figma
- Poppins Font
- Iconify Icons

## 📸 Deliverables

- Splash Screen Design
- Home Screen Design
- Food Details Screen
- Success Popup Screen
- Interactive Prototype
